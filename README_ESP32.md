# Generative Art
<br/>

Text based Generative Art on ESP32 with TT-GO display
<br/>
<br/>
The piece randomly chunks the words of a sentence and displays them for random intervals of time, adjusting the size of the words to fit the screen. Creates novel emphasis and phrasing that changes the interpretation of the sentence every time.

Rechargeable Lithium Polymer Battery (3.7 Volt, 150mAh) <br/>
